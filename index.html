Common grade combinations


Info

Calculate grades where minimum number is 4 and maximum number is 10. This is common in Finnish school system.

Grade is calculated linearly:

y = k(x - x_0) + y_0

Same in JS:

const result = ( ( 10 - 4 ) / ( max - min ) ) * ( points - min ) + 4;
